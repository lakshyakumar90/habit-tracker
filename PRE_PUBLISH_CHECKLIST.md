# Pre-Publish Checklist

## Assets

- [ ] App icon 1024x1024
- [ ] Adaptive icon foreground with safe-zone padding
- [ ] Adaptive icon background
- [ ] Monochrome icon for Android 13+
- [ ] Notification icon 96x96 white silhouette on transparent
- [ ] Splash image and matching background
- [ ] Favicon 48x48
- [ ] Play Store feature graphic 1024x500
- [ ] At least 4 screenshots per device class

## Configuration

- [ ] Bundle ID and package name finalized
- [ ] Version and build number set
- [ ] Unused permissions blocked
- [ ] Non-exempt encryption flag set correctly
- [ ] Privacy manifest configured where required
- [ ] Google services files in place (if used)
- [ ] EAS project ID set
- [ ] Runtime version policy configured

## Store Listings

- [ ] App name and short description finalized
- [ ] Full description finalized
- [ ] Category selected
- [ ] Content rating completed
- [ ] Privacy policy and terms URLs are live
- [ ] Support contact configured

## Quality

- [ ] Console logs removed or guarded by **DEV**
- [ ] Deep links tested
- [ ] Push notifications tested on real devices
- [ ] Cloud sync tested (enable/disable/re-enable)
- [ ] Widgets tested on real devices
- [ ] Small-screen and tablet QA complete
